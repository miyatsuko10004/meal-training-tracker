# M&T Optimizer (Meal & Training Optimizer) - 要件定義書

## 1. プロジェクト概要
日々の食事とトレーニング内容を記録し、その相関データを活用してトレーニングの質と身体作りを最適化するためのアプリケーション。

## 2. ターゲット
*   効率的に筋肉をつけたい、または減量したいトレーニー。
*   食事とトレーニングのデータを一元管理し、iPhoneとPCの両方からアクセスしたいユーザー。

## 3. 主要機能 (フェーズ 1: プロトタイプ)

### 3.1. 食事管理 (Meal Log)
*   **写真撮影/アップロード:** その場で撮影またはライブラリから選択。
*   **マクロ栄養素入力:** タンパク質(P)、脂質(F)、炭水化物(C)、総カロリーの入力。
*   **時間帯分類:** 朝、昼、晩、間食、トレーニング前後などのタグ付け。

### 3.2. トレーニング管理 (Training Log)
*   **セッション作成:** 「胸の日」「脚の日」などのタイトルと日付。
*   **種目記録:** 種目名、セット数、重量、回数(Reps)。
*   **強度管理:** RPE (主観的強度) の記録。

### 3.3. ダッシュボード・分析 (Analytics)
*   **目標設定:** ターゲット体重、1日の目標PFCバランスの設定。
*   **進捗グラフ:** 体重推移、摂取カロリーの1週間平均、部位別の週合計ボリューム。

### 3.4. 共通基盤
*   **デバイス対応:** iPhoneアプリ (Expo Native) および PCブラウザ (Expo Web)。

## 4. 技術スタック (案)
*   **Frontend:** React Native (Expo) + TypeScript + NativeWind (Tailwind CSS for React Native)
*   **Backend:** Node.js (Next.js API Routes or Express)
*   **Database:** SQLite / PostgreSQL (Prisma ORM)
*   **Hosting:** Vercel (Web/Backend), Expo Go (Mobile preview)

## 5. UI/UX コンセプト
*   **モバイル:** 片手で素早く入力できるUI。トレーニング中の入力を妨げないシンプルさ。
*   **PC:** 広い画面を活かしたデータ分析と長期的なトレンドの確認。
*   **デザイン:** ダークモード対応。清潔感のあるスポーティーなデザイン。
